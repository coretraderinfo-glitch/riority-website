# Riority Group Website

This repository contains the official website for **Riority Group (M) SDN BHD**, a private investment holding company established in 2012.

## Technologies Used
- **HTML5**
- **CSS3** (with Modern Layouts, Glassmorphism, and Animations)
- **Vanilla JavaScript** (for interactive components)

## Deployment
Ideally deployed on **Railway** or **GitHub Pages** for maximum uptime and performance.
